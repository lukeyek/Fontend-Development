Luke's Course & Career Prospect

Overview

Diploma in Applied AI and Analytics (DAAA), has become more important as there is an increase in the demand for data analysts and artificial intelligence practitioners. The course consists of the modules as listed below.

Modules:

Front-End Web Development

This module aims to equip students with the knowledge and skills in developing effective front-end web applications using Hypertext Markup Language (HTML) and Cascading Stylesheets (CSS).

Fundamentals of Programming

This module aims to help students pick up a programming language and learn how to solve and automate tasks through programming. Students will be taught programming fundamentals.

Fundamentals of Computing

This module aims to provide students with an understanding of computer networking concepts and hands-on sessions on with operating systems using Command-Line Interfaces.

Communicating for Project (Proposal) Effectiveness

This module aims to equip students with the skills to articulate and communicate ideas persuasively and to work effectively in teams. They will be taught to pitch ideas or concepts and to write proposals to an intended audience.

Critical and Analytical Thinking (CAT)

This module aims to equip students with skills in critical and analytical thinking, which includes the ability to evaluate different perspectives, articulate a point of view and support it with relevant and credible evidence.

Mathematics

This module equips students with knowledge in mathematics and analytical skills to solve problems related to infocomm technology. Topics include matrices, linear transformation, number systems, set theory, logic, Boolean algebra, techniques of counting and probability.

Education and Career Guidance 1: Personal Development

This module aims to help you discover your strengths, values and interests. It also supports you in making informed educational and career choices to achieve your career aspirations.
